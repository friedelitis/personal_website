3.2 Server Log Files

When you access this website, the web server automatically records information in so-called server log files, which your browser transmits automatically. This includes:

  • IP address of the requesting device
  • Date and time of access
  • Requested URL / page
  • Browser and operating system used (user agent)
  • Referrer URL (previously visited page)
  • Amount of data transferred
  • HTTP status code

This data is processed to ensure the secure and stable operation of the website and to detect and prevent attacks.

Legal basis: Art. 6(1)(f) GDPR (legitimate interest in the security and functionality of the website).

Storage duration: Server log files are typically deleted automatically by the hosting provider after 14 days. No personal evaluation of the data takes place.

Reclaim Hosting processes data on servers in the USA. The USA is a third country within the meaning of the GDPR. The transfer is based on:

Reclaim Hosting provides a Data Processing Agreement (DPA) with Standard Contractual Clauses. Further information: https://www.reclaimhosting.com/contracts/data-processing-agreement/
